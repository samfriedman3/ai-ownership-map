# Monitor proposals

The ownership monitor writes dated markdown files here (e.g. `2026-07-27.md`),
one per run, listing possible ownership changes it found via web search.

These are **drafts for human review**, not published data. A reviewer checks each
proposal against its sources, applies the good ones to `../ownership.json` (updating
the `verified` date on each edited row), and discards the rest. See the repo README
for the full workflow.
